## 技术栈

| 层级 | 技术选型 |
|------|---------|
| 管理端/大屏 | React 18 + Ant Design Pro + eCharts + 高德地图API |
| 司机App | 腾讯Kuikly (Kotlin Multiplatform) |
| 后端网关 | CloudWeGo Hertz |
| RPC微服务 | CloudWeGo Kitex + Netpoll |
| 数据库 | TiDB (分布式MySQL兼容) |
| 缓存 | Redis Cluster |
| 搜索 | Elasticsearch 8.x |
| 消息队列 | Apache RocketMQ |
| AI能力 | DeepSeek API + MediaPipe人脸关键点 + 疲劳识别模型 |
| 文件存储 | MinIO |
| 实时通信 | WebSocket |
| 边缘计算 | Android车载终端 (TensorFlow Lite) |

## 核心功能模块

1. **危险品运输路径规划** - A*算法 + 路权权重，自动避开人口密集区/隧道/桥梁/水源保护区
2. **驾驶员疲劳监控** - PERCLOS眼部闭合检测 + 哈欠检测 + 头部姿态分析
3. **实时监控与报警** - WebSocket推送 + 车辆图标状态 + 报警弹窗处置
4. **电子押运** - 运单全流程追踪 + 异常事件记录
5. **服务区停靠推荐** - 基于疲劳指数的智能停靠建议
6. **天气预警** - 恶劣天气实时告警 + 路径重规划
7. **车辆诊断** - OBD数据采集 + 故障预警
8. **驾驶评分** - 多维度行为评分体系
9. **紧急救援** - 一键SOS + 最近救援资源调度
10. **电子运单** - 危化品运输单据电子化
11. **区块链存证** - 关键数据上链 + 不可篡改溯源
